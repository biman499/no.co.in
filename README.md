# Biman_Photo_Lab

Welcome to **Biman_Photo_Lab**, a personal photography portfolio website showcasing the beauty of nature through the lens of Biman.

## Project Overview

This project is a static website designed to display a curated collection of photographs, focusing on landscapes, wildlife, and natural textures. It features a unique 3D rotating slider on the homepage to present highlights in an engaging way.

## Features

- **Home Page**: Features a 3D rotating image slider showcasing selected works with descriptions.
- **Gallery**: A grid layout displaying a collection of 25+ high-quality images.
- **About**: Information about the photographer and the vision behind the lab.
- **Contact**: A simple contact form and details for inquiries.

## Technologies Used

- **HTML5**: Semantic structure of the web pages.
- **CSS3**: Styling, including 3D transforms and animations for the slider.

## How to Run

1. Clone or download the repository.
2. Navigate to the project folder.
3. Open `index.html` in your preferred web browser.

## Pages

- `index.html`: Main landing page.
- `gallery.html`: Photo gallery.
- `about.html`: About the photographer.
- `contact.html`: Contact information.
